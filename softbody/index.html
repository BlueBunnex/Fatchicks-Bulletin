Softbody BBW Simulation in Blender

Introduction


1. source 2. source

As it turns out, simulating soft-bodies (and soft-bodied girls) isn't as easy as just clicking the "Add Soft-Body" button. I wanted to compile what I've learned about simulating BBWs into one tutorial to make it easier for new artists to pick up, or for more experienced artists to reference.

This tutorial will teach you how to simulate BBWs with Blender soft-bodies (make them squish against themselves and other objects, move with physics, and be animated). You will also learn how to render and export your simulation as a video.

This tutorial does not cover how to install Blender, model, or texture/create custom shaders. It also assumes you have basic experience in Blender already (such as moving objects and navigating the menus).

This tutorial is not finished. I'm still learning too!!
