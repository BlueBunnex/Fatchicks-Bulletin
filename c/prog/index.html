My custom programming lang specification

  • Only 1D arrays exist, do dimension control yourself
  • Structs, conceptually, are just arrays with keys
    • You can't resize arrays or assign new keys to structs after they are initialized
(int:foo, int[]:bar -> int):example_function = { return foo + bar[0]; } print (example_function 10 [10, 20, 30]);

Example of a struct factory.

(-> struct):make_point = { return [ int:x = 0, int:y = 0, (int:dx, int:dy ->):move = { x += dx; y += dy; } ]; } struct:point = make_point; point[move] 10 20; (* * this, in effect, changes nothing. It is here to show that * x and y in the initial declaration refer to specific points * in memory rather than "the value labeled x/y within the * struct I'm contained in" *) point[move] = { point[x] += dx; point[y] += dy; }
